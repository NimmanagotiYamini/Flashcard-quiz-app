Flashcard Quiz App

Overview

The Flashcard Quiz App is a Java Swing-based desktop application designed to help students study using flashcards. Each flashcard contains a question and an answer. Users can reveal answers, navigate through flashcards, and manage their own study material by adding, editing, and deleting flashcards.

Features

- Display flashcard questions.
- Show answers using the Show Answer button.
- Navigate between flashcards using Next and Previous buttons.
- Add new flashcards.
- Edit existing flashcards.
- Delete unwanted flashcards.
- Simple and user-friendly graphical interface.

Technologies Used

- Java
- Java Swing (GUI Framework)
- ArrayList Collection

How to Run

Using Command Prompt

1. Open the terminal or command prompt.
2. Navigate to the project directory.
3. Compile the program:

javac FlashcardQuizApp.java

4. Run the application:

java FlashcardQuizApp

Using an IDE

1. Open the project in your preferred IDE.
2. Run the "FlashcardQuizApp.java" file.
3. The application window will open.

Application Workflow

1. The application displays a flashcard question.
2. Click Show Answer to reveal the answer.
3. Use Next and Previous buttons to navigate between flashcards.
4. Click Add to create a new flashcard.
5. Click Edit to modify the current flashcard.
6. Click Delete to remove the current flashcard.
